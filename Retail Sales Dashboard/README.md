**Retail Sales Dashboard (2023)**


**Overview**
This project contains a **Power Bi dashboard** built using retail sales dataset with 1000+ transactions from 2023.
The dashboard provides insights into **sales performance, customer demographics, product categories, and seasonal trends**.

**Objectives**
The dashboard answers key business questions:
1. **How much are we selling overall?**
2. **Which product category drives revenue?**
3. **Who are our customers (gender & age group)?**
4. **When are sales happening (monthly trends)?**

**Key Features**

- **Top KPIs**: Total Revenue, Total Transactions, Average Transaction Value, Average Quantity per Transaction
- **Revenue by Product Category**: Electronics, Clothing, Beauty
- **Share of Transactions by Category** (Pie Chart)
- **Revenue by Gender** (Male vs Female comparison)
- **Revenue by Age Group** (Bucketed ranges: 18-25, 26-35, 46-60, 60+)
-  **Monthly Revenue Trend** (Seasonality Analysis)
